# My Dotfiles

## Requirements

- Git
- Curl

## Install

Install config tracking in your $HOME by running:

```sh
curl -Lks git.io/vHjm6 | /bin/bash
```

The expanded url is at:

https://raw.githubusercontent.com/hdnha11/dotfiles/master/.bin/dotfiles-install.sh
